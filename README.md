# Python Practice Problems

## 1. Basic Math & Formulas

1. **Oldest of Three:** User will input 3 ages. Find the oldest one.
2. **Temperature Conversion:** Write a program that will convert a Celsius value to Fahrenheit.
3. **Swap Two Numbers:** User will input 2 numbers. Write a program to swap the numbers.
4. **Sum of 3 Digits:** Write a program that will give you the sum of 3 digits.

6. **Euclidean Distance:** Write a program to find the Euclidean distance between two coordinates.
7. **Simple Interest:** Write a program to find the simple interest when the value of principal, rate of interest, and time period is given.
8. **Volume of Cylinder & Milk Cost:** Write a program to find the volume of a cylinder. Also find the cost when the cost of 1 liter milk is 40 Rs.
9. **Sum of Squares of Digits:** Write a program that will take three digits from the user and add the square of each digit.
10. **Sum of First n Numbers:** Write a program to find the sum of first $n$ numbers, where $n$ will be provided by the user (e.g., if $n = 10$, output should be `55`).
11. **Multiply Without `*` Operator:** Write a program that can multiply 2 numbers provided by the user without using the `*` operator.
12. **Compound Interest:** Write a program to find the compound interest.
13. **Compute $n + nn + nnn$:** Write a Python program that accepts an integer ($n$) and computes the value of $n + nn + nnn$.
